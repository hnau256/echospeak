Я изучаю новогреческий, я уже изучил лексику и грамматику для уровня B1, но говорю до сих пор медленно и подбирая слова. Поэтому хочу "загрузить" в свой языковой центр разнообразные языковые паттерны. Твоя задача сгенерировать 50 фраз на новогреческом языке по теме "Винительный падеж".
Главное требование: должны быть максимально полно покрыты разнообразные сочетания родов, склонений, чисел. Должны быть добавлены особо важные исключения. Нужно использовать как определённые, так и неопределённые артикли, а так же слова, которые могут их заменять.
Дополнительные требования: язык должен быть естественным, используй разнообразные но частотные глаголы, спряжения, лица, времена.

Ответ выдай в виде Json array пригодный для парсинга. Каждый элемент, которого должен парситься в:
@Serializable
data class Phrase(
    val phrase: String,
    val translation: String,
    val comment: String,
)

phrase - фраза на новогреческом
translation - перевод на русский
comment - объяснение грамматики и особенностей фразы
